# RestaurantOS Product Vision

## Version 1.0

RestaurantOS helps restaurants create more regulars through measurable hospitality. It combines operational data, financial performance, hospitality experiments, and AI-guided insights into one decision platform.

## Product Principles

- One source of truth.
- One-click refresh.
- Every report ends with a recommended action.
- Measure learning, not just performance.
- Calm, minimalist design.

## Core Modules

1. Executive Dashboard
2. Sales Intelligence
3. Beverage Intelligence
4. PMix & Menu Engineering
5. Hospitality Lab
6. Market Benchmarking
7. AI Manager Brief
8. Book of Experiments
