# RestaurantOS Constitution

## Version 1.0

> Purpose: To help restaurants create more regulars through measurable hospitality.

## First Principles

1. Hospitality before accounting.
2. Every metric must earn its place.
3. Every experiment creates knowledge.
4. Improvement never stops.
5. Trust is the foundation.

## The Five Questions

1. Are we healthy?
2. Are guests happier?
3. Are employees improving?
4. What did we learn?
5. What should we do next?

## Motto

**Not freebies. Seeds.**
