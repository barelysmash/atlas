# Architecture

## Version 1.0

## Layers

1. Data Layer
2. Model Layer
3. Presentation Layer
4. AI Layer

## Refresh Pipeline

1. Refresh data sources.
2. Validate row counts and totals.
3. Refresh model.
4. Recalculate KPIs.
5. Refresh dashboards.
6. Generate AI summary.
7. Log refresh status.

## Constraint

AI never invents data. Every insight must trace to validated metrics.
