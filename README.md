# Operational Intelligence Platform

> We help restaurants learn faster than they forget.

RestaurantOS is proof that hospitality can be improved through evidence without sacrificing humanity.

We do not optimize restaurants.

We help restaurants learn.

## Structure

- `atlas/` — reusable decision infrastructure engine
- `plugins/restaurant/` — restaurant domain plugin
- `restaurantos/` — first product implementation
- `restaurantos/fonda-san-miguel/` — first deployment target
- `docs/` — product, architecture, design, and operating documents
- `rfc/` — decision records and proposed changes
- `tests/` — validation and reconciliation tests
- `samples/` — sample datasets and templates
- `releases/` — versioned release notes

## Core Questions

Every release answers four questions:

1. What happened?
2. Why?
3. What should we do?
4. What did we learn?

## Motto

**Not freebies. Seeds.**

Every interaction plants trust, curiosity, confidence, and belonging. Revenue is the harvest.
